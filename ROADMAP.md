### Phase 0: OpenAPI Specification (Current Focus)

**Goal:** Create a comprehensive, accurate OpenAPI specification for VALR's API.

Before building any SDKs, the primary objective is to:

- Document all VALR API endpoints in OpenAPI 3.0 format
- Define request/response schemas accurately
- Include authentication methods and security schemes
- Ensure the specification is complete and validated

The OpenAPI specification will serve as the foundation for all future SDK generation.
