# Open VALR API Clients

[![License: MIT](https://img.shields.io/badge/License-MIT-brightgreen.svg)](./LICENSE)
[![Contributions Welcome](https://img.shields.io/badge/Contributions-Welcome-brightgreen)](./CONTRIBUTING.md)

_Community-driven SDKs for easy integration with the VALR crypto exchange API._

**Open VALR** is an open source, community-maintained collection of SDKs for
the [VALR exchange API](https://www.valr.com/).
It provides automatically generated clients (via the official or community OpenAPI spec)
for multiple programming languages.

> **Disclaimer:** This project is **not affiliated** with or endorsed by VALR.
> It is maintained independently by the open source community.
